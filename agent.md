Use NestJS with Fastify as the backend framework for building a high-performance and scalable server. Use Prisma ORM with PostgreSQL as the primary database for data management. Optimize all database access with efficient Prisma queries, including proper indexing, well-designed relations, and minimal over-fetching. Implement caching where necessary using Redis to improve performance for frequently accessed or computationally expensive queries. The system should be designed with scalability, modular architecture, and clean separation of concerns in mind to ensure maintainability and high performance under load.
